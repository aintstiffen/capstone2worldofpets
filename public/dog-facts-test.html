Pet Facts API Integration Test

This page demonstrates the Pet Facts toast notification functionality.

How it works:

  • A random pet fact (dog or cat) appears automatically on the first visit each day
  • Facts are randomly selected from either:
    • Dog API (for dog facts)
    • Cat Fact API (for cat facts)
  • The fact is stored in localStorage so it only shows once per day
  • Click buttons above to test different toast notifications manually
  • Click "Reset Storage" to clear localStorage and simulate a first visit
