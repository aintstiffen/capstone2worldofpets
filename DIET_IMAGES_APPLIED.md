## Adding More Diet Options

To add new diet options with images:

1. Go to Filament Admin Panel
2. Edit any pet breed
3. Scroll to "Common Diet" section
4. Click "Add diet item"
5. Enter diet name
6. **Paste image URL** in "Or Use Image URL" field
7. Add description
8. Save

Refer to `DIET_IMAGE_URLS.md` for more ready-to-use image URLs!
