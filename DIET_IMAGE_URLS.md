## How to Use

1. Go to your Filament admin panel
2. Edit a pet breed
3. Scroll to "Common Diet" section
4. Click "Add diet item"
5. Enter the diet name (e.g., "Working Breed Formula")
6. Paste the image URL in the "Or Use Image URL" field
7. Add the description
8. Save

The system will now accept both uploaded images and online URLs!
